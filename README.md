# Free Image Toolkit

This project is a free, browser-based image toolkit that allows users to crop, resize, enhance, and convert images.

## Features

-   **Versatile Editing**: Crop, resize, enhance, and convert images with an intuitive interface.
-   **Multiple Format Support**: Work with PNG, JPEG, WEBP, BMP, GIF, and HEIC for maximum flexibility.
-   **Fast Processing**: Edit images instantly in your browser, no sign-up or downloads required.
-   **Privacy-Focused**: All processing happens locally in your browser—no data is uploaded.

## Usage

1.  Open `index.html` in a web browser.
2.  Select a tool: Crop, Resize, Enhance, or Convert.
3.  Upload an image by dragging and dropping or by clicking the upload area.
4.  Use the controls to edit the image.
5.  Download the edited image.
